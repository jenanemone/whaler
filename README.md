# whaler
Data search tool for finding approach targets for client DCP. Search massive CSVs for repeated and appropriate looking clients for DCP.

The "Whaler" Workflow

Install a new library: python -m pip install <package>

See what's installed: python -m pip list

Update the "Manifest": python -m pip freeze > requirements.txt

Run a script: python main.py

Launch the Notebook server: python -m jupyter notebook
